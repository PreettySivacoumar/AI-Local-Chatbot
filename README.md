#  AI Local Chatbot (TinyLlama)

An intelligent local AI chatbot built using **TinyLlama (Local LLM)** and **Streamlit**.

This project allows users to interact with a locally running Large Language Model through a clean web interface.

---

## 🚀 Features

- 💬 Interactive chat interface
- 🧠 Powered by TinyLlama via Ollama
- 🌐 Runs completely locally (No cloud API required)
- ⚡ Fast and lightweight
- 🎨 Clean Streamlit UI

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Ollama
- TinyLlama (Local LLM)
